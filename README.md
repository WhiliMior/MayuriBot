# Mayuri Bot

一个以 [Graia Ariadne](https://github.com/GraiaProject/Ariadne) 框架为基础的 `时间线 Timeline TRPG规则` QQ跑团机器人

***

本bot没有除 `时间线TRPG规则` 之外的任何功能。

功能列表
---
### 1. 游戏功能
-- 角色
- 角色创建
- 多个角色管理
- 等级变更
- 角色Buff管理
- 角色状态记录

-- 检定
- 目标数值设置
- 检定
- 交涉检定

-- 武器
- 武器创建
- 武器管理

-- 战斗
- 战斗行动指令
- 战斗行动计算（不计入行动）
